    CSS (Cascading Style Sheets) is a language used to style and design websites. While HTML is used to create the structure and content of a web page,CSS is used to control how that content looks. It allows developers to change colors, fonts, spacing, layout, and much more, making websites more attractive and br CSS can be written directly in the HTML file or placed in a separate file. It also follows rules that determine which styles take priority when there are conflicts. Overall, CSS plays a crucial role in the appearance and design of web pages.


    Why it's called "Cascading":

    The "cascading" part means that styles can fall back or be overridden based on priority: Inline styles > Internal CSS > External CSS Specific selectors override general ones Later rules can override earlier ones if there's a confli


    Types of CSS:

    Inline CSS - inside HTML elements (bad for large projects) Internal CSS - in a style tag inside head External CSS - in a separate .css file linked to the HTML (best for maintainability)
